# Rock_Paper_Scissors

This project builds an Rock_Paper_Scissors, where the user plays against the computer.

### Prerequisites

Before you begin, you will need to have the following tools installed on your machine: Git. In addition, it is good to have an editor to work with code like VSCode

### Installing

```
# Clone this repository
$ git clone git@github.com:cinthiagodoi/Rock_Paper_Scissors.git

# Open the index.html file on your browser
```
